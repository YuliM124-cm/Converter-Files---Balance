Bagaimana keamanan data perusahaan dijaga?

👤 Login akun tim (Supabase Auth)

Setiap anggota tim login dengan akunnya sendiri — tidak ada akses tanpa autentikasi yang sah.

🧠 Diproses di memori

File dibaca ke memori, diproses, lalu berkas sementara dihapus segera — tidak dibiarkan tersimpan.

🚫 Tanpa penyimpanan data transaksi

Hasil konversi dikirim langsung ke Anda; tidak ada salinan data transaksi yang disimpan di server.

🔒 HTTPS otomatis

Vercel menyediakan enkripsi TLS otomatis untuk seluruh trafik dari browser ke server.
